# Genesis v24.0: "Reflexive Core" - Саморазвивающаяся Торговая Экосистема

![License](https://img.shields.io/badge/license-MIT-green.svg)
![Status](https://img.shields.io/badge/status-active-brightgreen.svg)
![Version](https://img.shields.io/badge/version-24.0-red.svg)

**Genesis** — это модульная, многопоточная, когнитивная алгоритмическая торговая система, разработанная для платформы MetaTrader 5. В ее основе лежит многоуровневый гибридный подход, управляемый **мета-обучающимся Оркестратором (RL Agent)**. Система динамически адаптируется к рыночным режимам, распределяет капитал между стратегиями и непрерывно самосовершенствуется через циклы R&D.

### ✨ Ключевые особенности и когнитивные слои

#### 🧠 1. Рефлексивное Ядро (RL Orchestrator)
*   **Мета-обучение:** RL-агент (на базе `Stable-Baselines3`) в реальном времени анализирует состояние системы (PnL, VaR, сентимент, режим рынка) и динамически распределяет капитал между стратегиями, максимизируя риск-скорректированную доходность.
*   **Режимная Адаптация:** Система классифицирует рынок на режимы ("Сильный тренд", "Рендж с высокой волатильностью") и применяет оптимальную матрицу распределения капитала для каждого режима.

#### 🌐 2. Когнитивный Слой (Knowledge Graph & NLP)
*   **Граф Знаний (Neo4j/SQLite):** Анализирует новостной фон (`NLP Processor` на базе `transformers`) для извлечения причинно-следственных связей (например, "ФРС повышает ставку" -> "USD укрепляется").
*   **Векторная База Данных (FAISS/Chroma):** Используется для семантического поиска (RAG) в новостном потоке, позволяя системе находить исторически схожие события и их влияние на рынок.
*   **Многофакторный Консенсус:** Финальный торговый сигнал формируется на основе взвешенного консенсуса: AI-прогноз, подтверждение классических стратегий, сентимент из Графа Знаний и On-Chain данные (для крипто).

#### 🧬 3. Адаптивный R&D (Genetic Programming)
*   **Самосовершенствование:** В фоновом режиме система ищет "слабые места" (например, режим рынка без прибыльной стратегии) и автоматически запускает эволюционный алгоритм (`Genetic Programming`) для создания, бэктестинга и валидации абсолютно новых торговых правил.
*   **Инкубация:** Новые стратегии проходят многоэтапную валидацию, включая 3-сегментный Walk-Forward анализ и стресс-тесты, прежде чем попасть в пул "живых" стратегий.

#### 🛡️ 4. Многоуровневое Управление Рисками
*   **Pre-Mortem Analysis:** Перед открытием сделки запускается симуляция Монте-Карло (GARCH) для оценки вероятности "катастрофического" убытка.
*   **Контроль Дрейфа (ADWIN):** Мониторинг ошибки прогноза AI-моделей. При обнаружении дрейфа концепции система автоматически снижает риск и запускает экстренное переобучение.
*   **Динамический Риск:** Размер риска на сделку адаптируется в зависимости от недавней прибыльности и текущей просадки.
*   **VaR-Хеджирование:** При превышении портфельного риска (Value-at-Risk) система может автоматически открыть хеджирующую позицию.

#### 🖥️ 5. Интерфейсы
*   **Современный GUI (PySide6):** Настольное приложение для оперативного контроля, управления рисками и визуализации Графа Знаний.
*   **Web Dashboard (FastAPI/WebSocket):** Удаленный мониторинг состояния системы, P&L, позиций и статуса потоков через веб-браузер.

### 🚀 Инструкции по установке и запуску

#### 1. Предварительные требования
*   Установленный **Python 3.10** или выше.
*   Установленный **Git**.
*   Установленный и запущенный торговый терминал **MetaTrader 5**.
*   (Опционально) Запущенный сервер **Neo4j** (если `graph_database.enabled: true`).

#### 2. Установка
1.  **Склонируйте репозиторий:**
    ```bash
    git clone <URL вашего репозитория>
    cd <имя папки проекта>
    ```
2.  **Создайте и активируйте виртуальное окружение:**
    ```bash
    # Windows
    python -m venv venv
    .\venv\Scripts\activate

    # macOS / Linux
    python3 -m venv venv
    source venv/bin/activate
    ```
3.  **Установите все необходимые зависимости:**
    ```bash
    pip install -r requirements.txt
    ```
    *Примечание: Установка PyTorch и LightGBM может занять несколько минут.*

#### 3. Конфигурация
1.  **Секретные данные (`configs/.env`):**
    *   Заполните ваши учетные данные для MT5 и API-ключи. **`MT5_LOGIN`, `MT5_PASSWORD`, `MT5_SERVER`, `MT5_PATH` являются обязательными.**
2.  **Основные настройки (`configs/settings.json`):**
    *   Настройте список `SYMBOLS_WHITELIST` и другие параметры.

#### 4. Запуск
1.  **Настройте терминал MT5:**
    *   Запустите MetaTrader 5 и войдите в свой счет.
    *   В меню `Сервис -> Настройки -> Советники` поставьте галочку **"Разрешить алгоритмическую торговлю"**.
    *   В окне "Обзор рынка" (`Ctrl+M`) добавьте все инструменты из `SYMBOLS_WHITELIST`.
2.  **Запустите приложение:**
    ```bash
    python main_pyside.py
    ```
    *После запуска система автоматически начнет синхронизацию истории, загрузку AI-моделей и инициализацию всех потоков.*
