# ⚡ Быстрый старт - Настройка GitHub для сборки

## 🚀 5 шагов к автоматической сборке

### Шаг 1: Запустите скрипт настройки

```batch
cd F:\MT5Qoder\MT5Projekt-Clean
setup_github.bat
```

### Шаг 2: Выберите пункт 1 (Инициализировать Git)

Скрипт создаст Git репозиторий и `.gitignore`

### Шаг 3: Выберите пункт 2 (Настроить удалённый репозиторий)

Введите URL вашего репозитория:
```
https://github.com/ВАШ_НИК/MT5Projekt-Clean.git
```

### Шаг 4: Выберите пункт 3 (Загрузить на GitHub)

Скрипт отправит файлы на GitHub

### Шаг 5: Выберите пункт 4 (Создать тег версии)

Введите версию: `13.0.0`

---

## ✅ Готово!

GitHub Actions автоматически запустит сборку.

**Перейдите в Actions:**
https://github.com/ВАШ_НИК/MT5Projekt-Clean/actions

**Через 30-50 минут:**
- ✅ Сборка завершится
- ✅ Артефакт будет доступен для скачивания
- ✅ Release будет создан

---

## 📥 Скачивание

1. Откройте вкладку **Actions**
2. Выберите последний запуск (зелёный)
3. Скачайте артефакт из раздела **Artifacts**
