## 🔍 Почему обучение не видно в логах?

### **Причина 1: Долгий интервал**

По умолчанию обучение запускается **раз в 24 часа**!

**Решение:**
```json
"TRAINING_INTERVAL_SECONDS": 3600  // 1 час
```

### **Причина 2: Ожидание 120 сек перед стартом**

```python
self.stop_event.wait(120)  # 2 минуты ожидания
```

**Решение:** Подождать 2 минуты после запуска системы

### **Причина 3: Пустой `latest_full_ranked_list`**

Торговый цикл еще не набрал данные

**Решение:** Подождать пока система наберет данные (5-10 минут)
