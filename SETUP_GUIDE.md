# 🚀 Genesis Trading System - Руководство по настройке

## 📋 Быстрый старт

### Шаг 1: Первый запуск

После распаковки архива `GenesisTrading_Portable_v1.0.0.zip`:

1. **Запустите `GenesisSetup.exe`** — мастер первоначальной настройки
2. Следуйте инструкциям мастера
3. После завершения настройки запустите `GenesisTrading.exe`

---

## 🔧 GenesisSetup.exe — Мастер настройки

### Что настраивает мастер:

#### Шаг 1: Пути к файлам и папкам
- **Путь к MetaTrader 5** (`terminal64.exe`)
- **Папка для баз данных** (по умолчанию: `database`)
- **Папка для логов** (по умолчанию: `logs`)
- **Папка для кэша AI-моделей** (опционально)

#### Шаг 2: Учетные данные MT5
- **Login** — номер счета
- **Password** — пароль
- **Server** — имя сервера (например, `Alpari-MT5-Demo`)

#### Шаг 3: API ключи (опционально)
- Finnhub — анализ рынка
- Alpha Vantage — финансовые данные
- NewsAPI — новости
- Polygon.io — рыночные данные
- Twelve Data — котировки
- FCS — финансовые данные
- FRED — экономические данные
- Neo4J — графовая база данных

#### Шаг 4: Торговые параметры
- **Риск на сделку (%)** — по умолчанию 0.5%
- **Макс. открытых позиций** — по умолчанию 5
- **Интервал торговли (секунды)** — по умолчанию 60

---

## 🎯 Автоматический запуск мастера

Если конфигурация не найдена или содержит ошибки, **мастер настройки запустится автоматически** при попытке запуска `GenesisTrading.exe`.

### Критерии автоматического запуска:

1. ❌ Файл `configs/settings.json` не найден
2. ❌ Отсутствуют обязательные параметры:
   - `MT5_LOGIN`
   - `MT5_PASSWORD`
   - `MT5_SERVER`
   - `MT5_PATH`
3. ❌ Указанный путь к MT5 не существует
4. ❌ Ошибка чтения конфигурации

## 🔧 Ручное редактирование конфигурации

Файл: `configs/settings.json`

  "MT5_PASSWORD": "your_password",

## ⚠️ Возможные проблемы

### Проблема 1: Мастер не запускается автоматически

**Решение:**
- Запустите `GenesisSetup.exe` вручную
- Проверьте, что `configs/settings.json` не существует или пуст

### Проблема 2: Ошибка "MT5 терминал не найден"

**Решение:**
1. Убедитесь, что MetaTrader 5 установлен
2. Найдите файл `terminal64.exe`
3. Укажите полный путь, например:
   ```
   C:/Program Files/MetaTrader 5/terminal64.exe
   ```

### Проблема 3: Ошибка доступа к папке

**Решение:**
- Запустите от имени администратора
- Проверьте права доступа к папке
- Используйте папки в директории пользователя

### Проблема 4: Конфигурация не сохраняется

**Решение:**
- Проверьте, что файл `settings.json` не открыт в другом приложении
- Убедитесь, что есть права на запись в папку `configs`
- Попробуйте запустить от имени администратора

## 🔄 Повторный запуск мастера

Если нужно изменить настройки:

1. Удалите файл `configs/settings.json`
2. Запустите `GenesisTrading.exe` или `GenesisSetup.exe`
3. Пройдите мастер заново

**ИЛИ**

Отредактируйте `configs/settings.json` вручную

## 📞 Поддержка

При возникновении проблем:

1. Проверьте логи в папке `logs/`
2. Убедитесь, что все пути указаны в формате Windows (`C:/...` или `C:\...`)
3. Проверьте, что MetaTrader 5 запускается независимо
4. Убедитесь, что у вас есть доступ к интернету для API запросов
