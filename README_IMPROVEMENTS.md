## 📊 Что будет изменено?

### Критические изменения (обязательно):

| Параметр | Было | Станет | Зачем |
|----------|------|--------|-------|
| RISK_PERCENTAGE | 2.0% | 0.5% | Снижение риска в 4 раза |
| MAX_OPEN_POSITIONS | 18 | 5 | Снижение экспозиции |
| STOP_LOSS_ATR_MULTIPLIER | 2.5 | 3.5 | Больше защиты |

| Model Profit Factor | Любой | ≥1.5 | Только прибыльные |
| Model Win Rate | Любой | ≥40% | Только надёжные |

### Новые компоненты:

1. **Model Validation Gate** - отклоняет убыточные модели
2. **Safety Monitor** - автоматическая остановка при просадке
3. **Walk-Forward Validation** - проверка на нескольких периодах
4. **Signal Quality Filter** - отклоняет слабые сигналы

## ⚠️ Важные предупреждения

### ❌ НЕ ДЕЛАЙ ЭТО:

1. **НЕ запускай на реальном счёте сразу** - минимум 30 дней на демо
2. **НЕ увеличивай риски обратно** - даже если кажется, что система работает
3. **НЕ отключай Safety Monitor** - он спасёт твой депозит
4. **НЕ принимай модели с плохими метриками** - лучше не торговать, чем терять

### ✅ ОБЯЗАТЕЛЬНО ДЕЛАЙ:

1. **Тестируй каждое изменение** - проверяй синтаксис и логи
2. **Мониторь систему ежедневно** - первые 2 недели особенно важны
3. **Веди журнал изменений** - записывай что и когда изменил
4. **Делай бэкапы** - перед каждым большим изменением

## 🆘 Что делать, если что-то сломалось?

### Проблема: Система не запускается

**Решение:**
1. Проверь логи: `logs/trading_system.log`
2. Найди последнюю ошибку (ERROR или CRITICAL)
3. Скажи AI: "Вот ошибка: [вставь ошибку]. Как исправить?"

### Проблема: Все модели отклоняются

**Это хорошо!** Значит старые модели были убыточными.

**Решение:**
1. Увеличь `TRAINING_DATA_POINTS` до 20000
2. Подожди 1-2 недели накопления данных
3. Попробуй другие символы (EURUSD вместо BITCOIN)

### Проблема: Система не торгует

**Это нормально!** Система стала консервативной.

**Проверь:**
1. Логи на `MODEL REJECTED` - если есть, модели плохие
2. Логи на `Signal rejected` - если есть, сигналы слабые
3. Подожди качественных сигналов (может занять дни)

### Проблема: Нужно откатить изменения

**Решение:**
```bash
# Откат конфигурации
git checkout configs/settings.json

# Откат кода
git checkout src/core/trading_system.py

# Удаление новых файлов
rm src/core/safety_monitor.py  # Linux/Mac
del src\core\safety_monitor.py  # Windows
```
