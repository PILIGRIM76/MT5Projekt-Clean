# ✅ ОБУЧЕНИЕ НАСТРОЕНО НА 30 МИНУТ

## 🎯 Изменения применены

### **Интервал обучения изменен:**

**БЫЛО:**
```json
"TRAINING_INTERVAL_SECONDS": 86400  // 24 часа
```

**СТАЛО:**
```json
"TRAINING_INTERVAL_SECONDS": 1800  // 30 минут
```

## 📁 Измененные файлы

### **`configs/settings.json`**

**Изменения:**
1. ✅ Строка 220: `TRAINING_INTERVAL_SECONDS: 1800`
2. ✅ Строка 299: `orchestrator.training_interval_seconds: 1800`
3. ✅ Строка 367: `orchestrator_settings.training_interval_seconds: 1800`

---

## 🚀 Что изменится

### **До:**
- Обучение запускалось **раз в 24 часа**
- Приходилось ждать целый день
- Редкое обновление моделей

### **После:**
- Обучение запускается **раз в 30 минут**
- Быстрое обновление моделей
- Частая адаптация к рынку

## 🔧 Как проверить

### **1. Перезапустить систему**

```bash
python main_pyside.py
```

### **2. Подождать 2 минуты**

Система наберет данные для обучения

### **3. Проверить логи**

Искать:
```
[R&D] Запуск цикла обучения...
--- НАЧАЛО R&D ЦИКЛА ---
--- R&D ЦИКЛ ЗАВЕРШЕН ---
```

### **4. Подождать 30 минут**

Убедиться что обучение повторяется
