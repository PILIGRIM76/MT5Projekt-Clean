## 🚀 Как установить

### Шаг 1: Запусти установщик

```powershell
.\GenesisTrading_Setup_v1.0.0.exe
```

Или двойной клик по файлу.

### Шаг 2: Следуй инструкциям

1. Выбери язык (English/Russian)
2. Прими лицензию
3. Выбери папку установки (по умолчанию: `C:\Program Files\Genesis Trading System`)
4. Выбери компоненты (опционально: Desktop icon)
5. Нажми "Install"
6. На последнем экране можешь поставить галочку "Launch Genesis Trading System"

### Шаг 3: Настрой конфигурацию

После установки настрой файл:
```
C:\Program Files\Genesis Trading System\_internal\configs\settings.json
```

Измени:
- `MT5_LOGIN` - твой логин MT5
- `MT5_PASSWORD` - твой пароль MT5
- `MT5_SERVER` - твой сервер MT5
- `MT5_PATH` - путь к terminal64.exe

## 🔧 Что делает установщик

1. **Копирует файлы** в `C:\Program Files\Genesis Trading System\`
2. **Создаёт папки:**
   - `database/` - для базы данных
   - `logs/` - для логов
   - `configs/` - для конфигураций
3. **Создаёт ярлыки:**
   - В меню Пуск
   - На рабочем столе (опционально)
4. **Копирует settings.json** из примера
5. **Регистрирует uninstaller** для удаления

## 🐛 Проблемы при установке

### Проблема: "Windows защитил ваш компьютер"

**Решение:**
1. Нажми "Подробнее"
2. Нажми "Выполнить в любом случае"

Это нормально для неподписанных приложений.

### Проблема: Антивирус блокирует установку

**Решение:**
1. Временно отключи антивирус
2. Установи программу
3. Добавь в исключения антивируса:
   - `C:\Program Files\Genesis Trading System\`
4. Включи антивирус обратно

### Проблема: Недостаточно прав

**Решение:**
Запусти установщик от имени администратора:
1. Правый клик на файле
2. "Запуск от имени администратора"
