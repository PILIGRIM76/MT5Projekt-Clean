## ⚙️ Настройки

### **В configs/settings.json:**

```json
{
  "SYMBOLS_WHITELIST": [
    "EURUSD", "GBPUSD", "USDJPY", "USDCAD",
    "AUDUSD", "USDCHF", "NZDUSD", "EURJPY",
    "GBPJPY", "EURGBP", "AUDJPY", "XAUUSD",
    "XAGUSD", "EURCHF", "CADJPY", "AUDNZD",
    "GBPAUD", "BITCOIN"
  ],
  "TOP_N_SYMBOLS": 18,
  "MAX_OPEN_POSITIONS": 18
}
```

### **Опции:**

| Параметр | Значение | Описание |
|----------|----------|----------|
| `SYMBOLS_WHITELIST` | Список | Все символы для торговли |
| `TOP_N_SYMBOLS` | 18 | Сколько символов ранжировать |
| `MAX_OPEN_POSITIONS` | 18 | Максимум открытых позиций |

## 🎯 Режимы торговли

### **Режим 1: Торговля по всем (18 символов)**

```json
{
  "TOP_N_SYMBOLS": 18,
  "MAX_OPEN_POSITIONS": 18
}
```

**Результат:**
- ✅ Обрабатываются все 18 символов
- ✅ Максимум 18 одновременных позиций
- ✅ Каждый символ проверяется на существующую позицию

### **Режим 2: Торговля по топ-N**

```json
{
  "TOP_N_SYMBOLS": 10,
  "MAX_OPEN_POSITIONS": 10
}
```

**Результат:**
- ✅ Топ-10 символов из рейтинга
- ✅ Максимум 10 позиций

### **Режим 3: Полная свобода**

```json
{
  "TOP_N_SYMBOLS": 0,  // 0 = все символы
  "MAX_OPEN_POSITIONS": 50
}
```

**Результат:**
- ✅ Все символы из whitelist
- ✅ До 50 одновременных позиций
