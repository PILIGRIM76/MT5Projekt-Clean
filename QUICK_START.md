# 🚀 Быстрый старт Genesis Trading System

## 📋 Требования

- ✅ Windows 10/11
- ✅ MetaTrader 5 установлен
- ✅ 8GB+ RAM
- ✅ Демо или реальный счёт MT5

---

## ⚙️ Первый запуск

При **первом запуске** программа автоматически создаст файл `configs\settings.json` с настройками по умолчанию.

### Шаг 1: Запусти программу

После установки запусти Genesis Trading System.

### Шаг 2: Настрой подключение к MT5

Открой файл настроек:
```
C:\Program Files\Genesis Trading System\configs\settings.json
```

Найди секцию MT5 и замени на свои данные:

```json
"MT5_LOGIN": "ВАШ_ЛОГИН",
"MT5_PASSWORD": "ВАШ_ПАРОЛЬ",
"MT5_SERVER": "ВАШ_СЕРВЕР",
"MT5_PATH": "C:/Program Files/ВашБрокер/terminal64.exe"
```

**Пример для Alpari:**
```json
"MT5_LOGIN": "12345678",
"MT5_PASSWORD": "MyPassword123",
"MT5_SERVER": "Alpari-MT5-Demo",
"MT5_PATH": "C:/Program Files/Alpari MT5/terminal64.exe"
```

### Шаг 3: Перезапусти программу

После сохранения настроек перезапусти Genesis Trading System.

**Это нормально, если первый запуск занимает 2-3 минуты!**

## ⚠️ Важные настройки безопасности

### Уже настроено (не меняй без необходимости):

```json
"RISK_PERCENTAGE": 0.5,           // Риск 0.5% на сделку
"MAX_OPEN_POSITIONS": 5,          // Максимум 5 позиций
"MAX_DAILY_DRAWDOWN_PERCENT": 5.0 // Стоп при просадке 5%
```

### Safety Monitor (автоматическая защита):

Система автоматически остановится если:
- ❌ Дневная просадка > 3%
- ❌ Просадка от пика > 5%
- ❌ 5 убыточных сделок подряд

**Это защищает твой капитал!**

## 📈 Первые 24 часа

### Что ожидать:

1. **Первые 2 часа:** Система сканирует рынок, НЕ торгует
2. **Через 24 часа:** Запустится R&D цикл (обучение моделей)

### Это НОРМАЛЬНО:

- ✅ Система почти не торгует первые дни
- ✅ Много "Signal rejected" в логах
- ✅ Много "MODEL REJECTED" в логах
- ✅ Нет открытых позиций

### Это НЕ НОРМАЛЬНО:

- ❌ Система крашится
- ❌ Нет логов `[SAFETY] Monitoring initialized`
- ❌ Открывается >5 позиций
- ❌ Модели принимаются с PF<1.5

## 🔧 Устранение проблем

### Проблема 1: "Не удалось подключиться к MT5"

**Решение:**
1. Проверь, что MT5 запущен
2. Проверь логин/пароль в `settings.json`
3. Проверь путь к MT5 в `settings.json`

### Проблема 2: "База данных заблокирована"

**Решение:**
1. Закрой все копии программы
2. Удали файл `database\trading_system.db-journal`
3. Перезапусти программу

### Проблема 3: Система не торгует

**Это НОРМАЛЬНО первые 24-48 часов!**

Система ждёт:
- ✅ Обучения моделей (24 часа)
- ✅ Качественных сигналов (высокие пороги)
- ✅ Подходящих рыночных условий

### Проблема 4: Ошибки в логах

**Проверь логи:**
```
logs\genesis_system.log
```

Если видишь ERROR - создай Issue на GitHub с логами.
